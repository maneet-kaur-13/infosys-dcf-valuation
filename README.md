# Business Valuation of Infosys Using Discounted Cash Flow (DCF) Model
### Transaction Advisory | Business Valuation | FY2022–FY2031

## Overview
A complete, analyst-grade DCF valuation of Infosys Limited (NSE: INFY) built 
using consolidated financial statements from FY2022–FY2026 and a 5-year FCFF 
forecast through FY2031.

## Methodology
**FCFF-based DCF** | WACC = 13.7% | TGR = 5.5%
FCFF = NOPAT + D&A − Capex − ΔNWC
